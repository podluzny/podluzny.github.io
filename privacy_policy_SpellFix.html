1. General Information

The ‚ÄúSpellFix‚Äù browser extension is designed to correct spelling and stylistic errors in text explicitly selected by the user using external Large Language Model (LLM) APIs.

We respect user privacy and do not collect personal data unnecessarily.

2. Data We Process

The extension may process:

  • Text explicitly selected by the user on a webpage.
  • Extension settings (API URL, API key, selected model, custom prompt).
  • Technical data required to send API requests.

The extension does NOT collect:

  • Browsing history
  • Visited websites list
  • Passwords
  • Cookies
  • Local files
  • Personal data without user action

3. How Data Is Used

Selected text:

  • Is temporarily sent to an external LLM API.
  • Is used solely to generate corrected text.
  • Is not stored by the extension developer.

User settings:

  • Are stored locally in the user's browser via chrome.storage.
  • Are not transmitted to the developer.
  • Are used only to communicate with the selected API.

4. Third-Party Services

If the user configures an external API provider (e.g., OpenAI, Anthropic, or other LLM services), selected text is transmitted directly to that provider.

The extension developer:

  • Does not receive
  • Does not store
  • Does not analyze user-submitted content

Users should review the privacy policy of their chosen API provider.

5. Data Storage

The extension does not operate remote servers.
All configuration data is stored locally in the user's browser.

6. Security

  • Does not include analytics tools
  • Does not include tracking scripts
  • Does not perform hidden data collection
